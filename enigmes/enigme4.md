# Enigme 4 : La chambre des échos

## Trois niveaux de difficultés : les élèves doivent se reconnaître dans tous les cas.

    - Premier niveau : les LLM inventent indépendamment une réponse
    - Deuxième niveau : les LLM essaient d'imiter "des élèves de seconde"
    - Troisième niveau : les LLM essaient d'imiter "des élèves de seconde" et ont accès aux prompts des deux élèves pour faciliter l'imitation
